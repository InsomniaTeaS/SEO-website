# SEO website

This is a multi-page class project about search engine optimization. I built it with HTML and CSS to practice page structure, navigation, responsive design, forms, image placement, and organizing related content across several pages.

## Pages

- `index.html` - short introduction to SEO
- `whatandtypes.html` - explanation of SEO and major SEO categories
- `bestpractices.html` - basic SEO recommendations
- `credits.html` - sources used for written content and images
- `form.html` - sample HTML form using text, date, select, radio, checkbox, textarea, submit, and reset controls

## Styling

The shared stylesheet is located at `styles/SEO.css`. It handles the site colors, navigation, page layout, images, form controls, and responsive behavior.

## What I practiced

- semantic HTML page structure
- shared navigation across multiple pages
- responsive layouts with media queries
- form inputs and required fields
- external stylesheets
- image alignment and sizing
- source attribution
